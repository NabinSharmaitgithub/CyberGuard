Internal Tools

Available Tools

  • bruteforce: SSH bruteforcer (Go)
  • chat: Encrypted chat application (Go)
  • findConnected: Discover active hosts (Go)
  • portScanner: Port scanner (Go)
  • siteMapper: Site mapper (Go)
  • sniffer: Packet sniffer (Go and Python)
  • trojanCreator: Trojan creator (Go)
  • userRecon: User reconnaissance (Go)

How to Use

To use these tools, you need to compile and run them manually from the command line.

For Go programs, you need to have Go installed. You can then run them using go run <file>.go.

For Python programs, you can run them using python <file>.py.

Note: Some of these tools may require root privileges to run.
